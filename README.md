### Self Host

- Clone the Repo
```
git clone https://github.com/Sadew451/SDMegaDL-Bot
```
- Enter the directory
```
cd MegaDL-Bot
```
- Install Requirements
```
pip3 install -r requirements.txt
```
- Run the Bot
```
python3 main.py
```
